== Description ==

🚀 [GitHub Repository](https://github.com/nhrrob/nhrrob-ai-assistant) – Found a bug or have a feature request? Let us know!

NHR AI Developer Assistant is a powerful tool that brings the power of Anthropic's Claude AI directly into your WordPress dashboard. 

### ✨ Key Features
- **AI Chat Assistant** – A modern floating widget available across all admin pages.
- **Safety First** – Mandatory validation of all code generated by AI before execution.
- **Rollback System** – Easily undo any change made by the AI with a single click.
- **Developer Focused** – Clean, modern React-based admin interface.

== Installation ==

1. Upload the NHR AI Developer Assistant plugin to your blog.
2. Activate it.
3. Go to AI Developer > Settings and enter your Claude API Key.
